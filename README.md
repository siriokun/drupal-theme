# rio
Basic theme for Drupal 11

## Installation Steps

1. Copy this repository to the `web/themes/rio` directory.
2. Run `ddev drush en rio -y` to enable the theme.
3. Go to directory `web/modules` and run `git clone git@github.com:siriokun/drupal-module.git rio_blocks` to clone the module.
4. Run `ddev drush en rio_blocks -y` to enable the module.
